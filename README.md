## 🏗 Architecture

Frontend: React + Tailwind  
Backend: Supabase (Postgres + Auth + Storage)  
Payments: Paystack  
Server Logic: Edge Functions  

Flow:
User → Enroll → Payment → Verification → Course Unlock → Certificate
